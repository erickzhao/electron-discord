## Base

> `base-secret` secret name

| -------------------- | --------------------------------- |

| `MYSQL_HOST`         | the url to the mysql db           |
| `MYSQL_USENRAME`     | the username which connects to db |
| `MYSQL_PASSWORD`     | the password                      |
